# 📚 북끼 Bookkki

## 📋 프로젝트 소개

Bookkki는 사용자가 읽은 책을 공유하고 기록하는 웹 어플리케이션입니다. 네이버 도서 API을 활용하여 도서를 제공하는 기능을 구현하였습니다.
